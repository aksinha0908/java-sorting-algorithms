# Java sorting algorithms
The console application shows performance of various sorting algorithms like Bubble Sort, Quick Sort etc.

## Technology
- Maven
- Java 11 (AWS Coretto)
- design pattern- Factory
- Enum Types
- Thread
- Testing
    - JUnit 4.12
    - Mockito 1.9.5
    - PowerMock 1.6.4
